# Tips for a successful planning stage

## Draw your wireframe - DONE

- Tools: Figma, Ok so, Excalidraw...
- Purpose: to see our app from the user's point of view.

## Plan the user flow - DONE

- Tools: Figma, Ok so, Excalidraw...
- Purpose: to see how the user will navigate our app

## Plan your database schema - DONE

- Tools: drawsql, Figma
- Purpose: to organise our data and to have a clearer idea of the data you need to manipulate.
- Questions: how many tables do I need? What is the relationship between the tables?
- Tip: use the Clerk userId as the primary key in the users table.

## Set up your database tables using your schema - DONE

- Tools: SQL editor on Supabase.
- Purpose: to create our tables and seed dummy data.

## Set up your Next app

- Tools: Next.js
- Purpose: to have a working "engine" with CR(UD) tasks and routes for our app and a connection to our database, and to have an interface to interact with the user.
